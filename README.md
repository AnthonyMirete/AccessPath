# AccessPath

AccessPath is a mobile application designed to help users find and share accessible routes for people with mobility challenges. The application will allow users to view accessible paths, report accessibility issues, and contribute route information.

## Technologies
- Android (Android Studio)
- GitHub for version control
- Database for storing accessibility route information

-Set up GitHub repo + development environment
-Create initial UI wireframes 
-Define database schema for accessibility reports
